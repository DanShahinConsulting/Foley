## Foley 

Simple drag and drop soundboard


- supports .mp3 .wav and .ogg
- stores sounds in localstorage
- keyboard shortcuts
- midi controller support

### Prebuilt App
- [MacOS](dist/Foley-darwin-x64-1.0.0.zip)




### Developer installation

```markdown

- `yarn install`
- `npm start`
```
